================================================================================
                        HOW TO USE THE WEBSITE
================================================================================

1. OPEN THE WEBSITE:
   Option A: Double-click index.html
   Option B: python3 -m http.server 8000 (then visit localhost:8000/index.html)
   Option C: Upload to web server

2. CUSTOMIZE FOR YOUR SALON:
   - Edit salon name: Replace "SALONE" with your name
   - Update contact info: Phone, email, address, hours in footer
   - Replace images: Put your salon photos in /images folder
   - Modify colors: Edit CSS variables in styles.css
   - Update services: Edit service descriptions
   - Add testimonials: Add more testimonial-slide divs
   - Update FAQ: Edit questions and answers

3. CONNECT THE CONTACT FORM:
   Currently: Form validation works, shows success message
   Option 1: Use Formspree (formspree.io)
   Option 2: Use Basin (usebasin.com)
   Option 3: Use EmailJS
   Option 4: Connect to your backend

4. DEPLOY:
   - Vercel: git push (auto-deploys)
   - Netlify: Upload files directly
   - Traditional hosting: FTP upload
   - GitHub Pages: Push to gh-pages branch
